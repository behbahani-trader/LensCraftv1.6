راهنمای اجرای برنامه روی ویندوز

مراحل کامل برای اجرای برنامه به صورت محلی روی کامپیوتر شما.

مقدمه

این برنامه یک اپلیکیشن وب است و روی هر سیستم عاملی که مرورگر وب داشته باشد، از جمله ویندوز، اجرا می‌شود. برای اجرای آن روی کامپیوتر شخصی خود، نیاز به یک محیط توسعه کوچک دارید که با چند مرحله ساده قابل راه‌اندازی است. این راهنما شما را قدم به قدم همراهی می‌کند.

مرحله ۱: نصب Node.js

ابتدا باید Node.js را نصب کنید. این ابزار به ما اجازه می‌دهد تا وابستگی‌های برنامه را مدیریت کرده و یک سرور توسعه محلی را اجرا کنیم.

به وبسایت رسمی Node.js بروید و نسخه LTS (Recommended For Most Users) را دانلود و نصب کنید. مراحل نصب بسیار ساده و شبیه به سایر نرم‌افزارهاست.

رفتن به سایت Node.js

مرحله ۲: باز کردن ترمینال و رفتن به پوشه پروژه

ترمینال ویندوز را باز کنید. می‌توانید کلمه PowerShell یا cmd را در منوی استارت جستجو کنید. سپس با استفاده از دستور cd به پوشه‌ای که فایل‌های برنامه در آن قرار دارند بروید.

# مثال: اگر پروژه شما در دسکتاپ است
cd C:\Users\YourUserName\Desktop\project-folder

مرحله ۳: نصب وابستگی‌ها

در ترمینال (که در مرحله قبل باز کردید)، دستور زیر را برای نصب تمام پکیج‌های مورد نیاز برنامه اجرا کنید. این فرآیند ممکن است چند دقیقه طول بکشد.

npm install

مرحله ۴: اجرای برنامه

پس از اتمام نصب، دستور زیر را برای اجرای سرور توسعه محلی اجرا کنید.

npm run dev

شما خروجی‌ای شبیه به این خواهید دید که آدرس محلی برنامه را نشان می‌دهد:

VITE v5.2.0  ready in 380 ms

➜  Local:   http://localhost:5173/
➜  Network: use --host to expose
➜  press h + enter to show help

مرحله ۵: مشاهده در مرورگر

در نهایت، مرورگر وب خود (مانند کروم یا اج) را باز کرده و آدرس http://localhost:5173 را در نوار آدرس وارد کنید. برنامه باید در مرورگر شما نمایش داده شود.

تبریک می‌گوییم! 🎉 شما با موفقیت برنامه را روی ویندوز اجرا کردید. تا زمانی که ترمینال باز باشد و سرور در حال اجرا باشد، برنامه در دسترس خواهد بود.
